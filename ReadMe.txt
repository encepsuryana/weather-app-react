Weather App

Introduction
the Weather App build form React js and Tailwind CSS for React, this app will show the data weather like Base the Location, Current time, Icon Weather, Degree Status and many more

Feature
- React js
- Tailwind CSS
- weatherapi.com

How to
1. To use or run this project, you must install node.js application in your machine, for the installation follow this link https://nodejs.org/en/download/

2. Open the terminal or Command Promp (CMD) and change the directory to project directory

3. In the project directory, you should run code `npm install` for installation the module to run this application

4. After installation Complete, you can run `npm run start` or `npm run`

5. The project will be Open to this URL http://localhost:3000 to view Application in your browser.

6. The application will show up in your browser and then you can search city or country for use the application
